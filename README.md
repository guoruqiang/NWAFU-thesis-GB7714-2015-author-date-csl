> **写在前面的吐槽**
> 西北农林科技大学开题报告及论文要求使用著者-出版年制（Author-Date）格式，并且要求中英文混排——中文文献显示"等"，英文文献显示"et al."。然而，**Zotero 官方样式库和中文社区都没有找到现成的样式支持这种混排格式。可能是我没有找到吧，不管了，自己动手丰衣足食吧**！😤
>
> 标准的 GB/T 7714—2015 样式要么全是"等"，要么全是"et al."，无法在同一篇文档中智能区分中英文文献。无奈之下，只能自己动手改了一个。如果你也遇到同样的问题，这个样式应该能帮到你！🎉

适用于西北农林科技大学硕士、博士研究生开题报告及论文的 Zotero 引文样式（CSL）。

## ⭐ 核心亮点

### 🌟 真正的中英文混排支持（自动识别）

**这是本样式最大的特色！** 不同于其他样式，本样式能够：

- ✅ **自动根据文献语言字段智能识别**中英文文献
- ✅ **中文文献自动显示**："和"、"等"
- ✅ **英文文献自动显示**："and"、"et al."
- ✅ **无需手动切换**，一次设置，全文生效
- ✅ **支持混合引用**，同一段落可同时引用中英文文献

**效果对比：**

| 其他样式 | 本样式 |
|---------|--------|
| 全部显示"等" 😞 | 中文显示"等"，英文显示"et al." 😊 |
| 全部显示"et al." 😞 | 根据语言自动切换 😊 |
| 需要手动编辑 😞 | 自动识别，无需干预 😊 |

## 📋 样式信息

- **引用格式**：著者-出版年制（Author-Date）
- **标准依据**：GB/T 7714—2015《信息与文献 参考文献著录规则》
- **适用范围**：西北农林科技大学硕博士开题报告及毕业论文
- **软件支持**：Zotero 文献管理软件
- **特色功能**：中英文混排自动识别

#### 📝 引用格式示例

**正文引用：**
- 2位作者（中文）：`(张三和李四，2021)`
- 3位及以上作者（中文）：`(张三等，2021)`
- 2位作者（英文）：`(Smith and Jones, 2021)`
- 3位及以上作者（英文）：`(Smith et al., 2021)`

**参考文献表：**
```
陈明, 李华, 2020. 中国农业现代化研究. 农业经济问题, 45(3): 123-145.
王五, 张三, 刘六, 等, 2019. 农村电商发展模式研究. 中国农村经济, 40(2): 50-60.
张三, 李四, 2021. 农业供给侧改革研究. 北京: 中国农业出版社: 1-200.
BROWN A, DAVIS B, 2020. Agricultural economics. New York: Academic Press: 1-300.
SMITH J, JONES M, 2021. Rural development. Journal of Rural Studies, 45(3): 123-145.
WILSON M, TAYLOR R, ANDERSON K, et al., 2019. Food security. Food Policy, 40(2): 50-60.
```

**排序规则：**
- ✅ **先中文后英文**：中文文献在前，英文文献在后
- ✅ **同语言内按作者排序**：中文按拼音，英文按字母
- ✅ **同作者按年份排序**：从早到晚
- ✅ **无编号**：不显示 [1]、[2] 等序号
- ✅ **悬挂缩进**：首行顶格，后续行缩进

## 🆚 与标准 GB/T 7714—2015 样式的区别

本样式基于社区维护的 `GB/T 7714—2015（顺序编码，双语，姓名不大写，无 URL、DOI）` 样式修改而来，主要差异如下：

| 特性 | 标准样式（bak.csl） | 本样式 |
|------|-------------------|--------|
| **引用格式** | 顺序编码制（Numeric） | 著者-出版年制（Author-Date） |
| **正文引用** | 上标序号 `[1,2]` | 括号作者年份 `(张三，2021)` |
| **参考文献排序** | 引用顺序 | 先中文后英文，按作者-年份排序 |
| **参考文献编号** | 有编号 `[1]` | 无编号 |
| **参考文献缩进** | 第二字段对齐 | 悬挂缩进 |
| **URL/DOI 显示** | 显示 | 不显示 |
| **访问日期显示** | 显示 | 不显示 |
| **OL 标识** | 显示 `[J/OL]` | 不显示 |
| **中英文混排** | 支持（单一 layout） | 增强支持（多 layout） |
| **语言本地化** | 仅中文 locale | 中文 + 英文 locale |
| **引用分隔符** | 英文逗号 `,` | 中文分号 `；`（中文）/ 英文分号 `;`（英文） |
| **字段分隔符** | 英文逗号 `,` | 中文逗号 `，`（中文）/ 英文逗号 `,`（英文） |
| **适用场景** | 通用学术论文 | 西北农林科技大学硕博士开题报告及毕业论文 |

### 主要修改内容

1. **引用格式转换**：从顺序编码制改为著者-出版年制
2. **参考文献排序**：先中文后英文，同语言内按作者-年份排序
3. **参考文献格式**：
   - 移除编号（无 [1]、[2] 等）
   - 改为悬挂缩进（首行顶格，后续行缩进）
   - 不显示 URL、DOI、访问日期
   - 不显示 OL 标识（[J/OL]、[M/OL] 等）
4. **多语言布局**：实现真正的 per-item locale selection
   - 为 `<citation>` 和 `<bibliography>` 添加多个 `<layout locale="xx">` 块
   - 英文文献使用英文标点和术语
   - 中文文献使用中文标点和术语
5. **语言本地化增强**：添加完整的英文 locale 定义
6. **标点符号优化**：根据文献语言自动选择中英文标点

## 📥 安装方法：直接下载安装

1. 下载 `西北农林科技大学硕博士开题及论文引文格式（GB-T-7714—2015）.csl` 文件
2. 打开 Zotero → 编辑 → 首选项 → 引用 → 样式
3. 点击 `+` 按钮，选择下载的 `.csl` 文件
4. 安装完成


## 🚀 快速开始

### 第一步：设置文献语言字段（关键！）

**⚠️ 这是实现中英文混排自动识别和正确排序的必要步骤！** 样式会根据每条文献的语言字段：
- 自动选择对应的术语（"等" vs "et al."）
- 自动排序（中文在前，英文在后）

| 文献类型 | 语言字段设置 | 注意事项 |
|---------|------------|---------|
| 英文文献 | `en` 或 `en-US` | ⚠️ 必须小写，不能是 `english` |
| 中文文献 | `zh` 或 `zh-CN` | ⚠️ 必须小写，不能是 `中文` |

**批量设置方法：**
1. 在 Zotero 中按标题或作者排序
2. 选中所有英文文献（Ctrl/Shift 多选）→ 在"语言"字段输入 `en`
3. 选中所有中文文献 → 在"语言"字段输入 `zh`

**语言字段的作用：**
- ✅ 控制术语显示（"等" vs "et al."）
- ✅ 控制排序顺序（中文在前，英文在后）
- ✅ 控制标点符号（中文标点 vs 英文标点）

### 第二步：在 Word 中使用

1. Zotero 工具栏 → Document Preferences → 选择本样式
2. 插入引用
3. 修改文献语言字段后，点击"刷新"按钮更新引用

## ⚠️ 常见问题

### Q1: 英文文献仍显示"等"而不是"et al."？
**原因**：文献的语言字段未设置或设置错误
**解决**：检查该文献的"语言"字段，确保填写 `en`（小写），然后刷新 Word 引用

### Q2: 样式验证时出现警告？
**原因**：本样式使用多语言布局技术，不符合标准 CSL 1.0 规范
**说明**：这是正常的！Zotero 的 citeproc-js 完全支持此语法，不影响使用

### Q3: 所有文献都显示同一种语言的术语？
**原因**：文献的语言字段未正确设置
**解决**：
- 英文文献设置为 `en`（不是 `english`）
- 中文文献设置为 `zh`（不是 `中文`）
- 未设置语言字段的文献将使用默认格式（中文）

### Q4: 参考文献没有按照"先中文后英文"排序？
**原因**：文献的语言字段未设置或设置错误
**解决**：
- 确保中文文献的语言字段设置为 `zh` 或 `zh-CN`
- 确保英文文献的语言字段设置为 `en` 或 `en-US`
- 刷新 Word 引用后，参考文献会自动按照"先中文后英文"排序

### Q5: 参考文献显示了 URL、DOI 或访问日期？
**说明**：本样式已配置为不显示这些信息
**检查**：确保使用的是最新版本的 CSL 文件
